### **1.** ***\*代码编辑与智能辅助\****

表格



| 功能                                       | 快捷键                 | 说明                                                    |
| :----------------------------------------- | :--------------------- | :------------------------------------------------------ |
| **基本代码补全**                           | `Ctrl + Space`         | 补全变量、函数、模块等                                  |
| **智能类型感知补全**                       | `Ctrl + Shift + Space` | 更精准的上下文补全（比如只显示字符串方法）              |
| **快速修复 / 意图操作**                    | `Alt + Enter`          | 自动导入、创建函数/类、修复错误、优化代码等（超常用！） |
| **生成代码（构造函数、getter/setter 等）** | `Alt + Insert`         | 在类中快速生成 `__init__`, `repr`, `property` 等        |
| **包围代码（with, try, if 等）**           | `Ctrl + Alt + T`       | 选中代码后包裹结构                                      |
| **注释行**                                 | `Ctrl + /`             | 切换单行注释                                            |
| **注释块**                                 | `Ctrl + Shift + /`     | 切换多行注释                                            |
| **格式化代码**                             | `Ctrl + Alt + L`       | 自动按 PEP8 或项目风格排版                              |
| **优化导入**                               | `Ctrl + Alt + O`       | 删除未使用的 import，排序 import                        |

### **2.** ***\*导航与查找\****

表格



| 功能                        | 快捷键                          | 说明                             |
| :-------------------------- | :------------------------------ | :------------------------------- |
| **跳转到定义**              | `Ctrl + B` 或 `Ctrl + 鼠标左键` | 查看函数/类/变量定义             |
| **查看用法（Find Usages）** | `Alt + F7`                      | 查找当前符号在项目中所有使用位置 |
| **快速跳转到任意文件**      | `Ctrl + Shift + N`              | 输入文件名快速打开               |
| **跳转到类**                | `Ctrl + N`                      | 输入类名快速定位                 |
| **跳转到符号（函数/变量）** | `Ctrl + Shift + Alt + N`        | 全局搜索函数或变量               |
| **最近文件切换**            | `Ctrl + E`                      | 打开最近编辑的文件列表           |
| **最近编辑位置**            | `Ctrl + Shift + E`              | 查看最近修改的位置               |
| **返回上一位置**            | `Ctrl + Alt + ←`                | 类似浏览器“后退”                 |
| **前进到下一位置**          | `Ctrl + Alt + →`                | “前进”                           |

------

### **3.** ***\*重构（不止重命名！）\****

表格



| 功能                         | 快捷键           | 说明                      |
| :--------------------------- | :--------------- | :------------------------ |
| **重命名（变量/函数/类等）** | `Shift + F6`     | 安全重命名所有引用        |
| **提取变量**                 | `Ctrl + Alt + V` | 将表达式提取为局部变量    |
| **提取方法**                 | `Ctrl + Alt + M` | 将选中代码封装成新函数    |
| **提取字段（类属性）**       | `Ctrl + Alt + F` | 把局部变量转为 `self.xxx` |
| **内联（反向提取）**         | `Ctrl + Alt + N` | 将变量/函数内联回原处     |
| **移动（Move）**             | `F6`             | 移动类/函数到其他文件     |
| **安全删除**                 | `Alt + Delete`   | 删除前检查是否被引用      |

### **4.** ***\*调试（Debug）\****

表格



| 功能                      | 快捷键       | 说明                       |
| :------------------------ | :----------- | :------------------------- |
| **调试运行**              | `Shift + F9` | 以调试模式运行当前脚本     |
| **继续执行（Resume）**    | `F9`         | 跳到下一个断点             |
| **单步跳过（Step Over）** | `F8`         | 执行当前行，不进入函数内部 |
| **单步进入（Step Into）** | `F7`         | 进入当前行调用的函数       |
| **跳出函数（Step Out）**  | `Shift + F8` | 从当前函数返回上层         |
| **运行到光标处**          | `Alt + F9`   | 快速跳到指定行             |

### **5.** ***\*版本控制（Git）\****

表格



| 功能             | 快捷键                  | 说明                  |
| :--------------- | :---------------------- | :-------------------- |
| **提交代码**     | `Ctrl + K`              | 打开提交窗口          |
| **推送代码**     | `Ctrl + Shift + K`      | Push 到远程仓库       |
| **查看本地变更** | `Alt + `（反引号）      | 快速查看未提交的修改  |
| **比较文件差异** | 选中文件后按 `Ctrl + D` | 查看与上次提交的 diff |

### **6.** ***\*窗口与工具\****

表格



| 功能                | 快捷键                           | 说明                               |
| :------------------ | :------------------------------- | :--------------------------------- |
| **显示/隐藏侧边栏** | `Alt + 1`                        | 项目结构面板                       |
| **显示终端**        | `Alt + F12`                      | 内置终端（非常常用）               |
| **运行当前脚本**    | `Shift + F10`                    | 快速运行                           |
| **停止运行**        | `Ctrl + F2`                      | 终止当前程序                       |
| **搜索 everywhere** | `Double Shift`（连按两次 Shift） | 全局搜索：文件、命令、设置、动作等 |

### **其他相关行操作快捷键（Windows）：**

表格



| 操作                     | 快捷键                                 |
| :----------------------- | :------------------------------------- |
| **删除当前行**           | `Ctrl + Y`                             |
